## Example

Given the following arbitrary blocks in `fixtures/c.txt`:

```handlebars
{{#definedoc "foo" a.b}}
This is content foo
{{/definedoc}}

{{#definedoc "bar" a.c}}
This is content bar
{{/definedoc}}

{{#definedoc "baz" a.d}}
This is content baz
{{/definedoc}}
```

The following:

```js
var str = fs.readFileSync('fixtures/c.txt', 'utf8');
console.log(blocks(str));
```

Results in:

```js
{ definedoc:
   { foo:
      { args: [ 'a.b' ],
        content: 'This is content foo',
        orig:
         [ '{{#definedoc "foo" a.b}}',
           'This is content foo',
           '{{/definedoc}}' ] },
     bar:
      { args: [ 'a.c' ],
        content: 'This is content bar',
        orig:
         [ '{{#definedoc "bar" a.c}}',
           'This is content bar',
           '{{/definedoc}}' ] },
     baz:
      { args: [ 'a.d' ],
        content: 'This is content baz',
        orig:
         [ '{{#definedoc "baz" a.d}}',
           'This is content baz',
           '{{/definedoc}}' ] } } }
```
